# Technical Report

The technical report is available in the `docs/` folder as HTML, or can be compiled from LaTeX sources.

## For PDF Version

If you have a LaTeX environment, compile:
```
pdflatex deadlock_defender_report.tex
```

Or view the HTML version at: `docs/project_report.html`
